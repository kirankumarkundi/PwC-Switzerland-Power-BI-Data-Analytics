# PwC-Switzerland-Power-BI-Data-Analytics


The PwC Switzerland project showcases the use of digital tools in data visualization, automation, and data cleansing to tackle business challenges. It includes Power BI dashboards on Call Centre Trends, Customer Retention, and Diversity & Inclusion. These dashboards offer actionable insights into business operations, enhancing efficiency, customer loyalty, and promoting inclusivity. The project aims to support informed decision-making and identify areas for improvement and innovation through meticulous data analysis and visualization.
